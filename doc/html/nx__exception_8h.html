Detailed Description

Abstract Exception Interface for all thrown NX errors.

NX-SDK v1.0.0 supports

  • Throws Simple exception of the type in
    • C++ - std::logic_err
    • Python - String Object.

NX-SDK v1.5.0 supports

  • Throws Simple exception of the type as version v1.0.0,
    • C++ - std::logic_err
    • Python - String Object.

  • Also Throws Advanced exeception of the type NxException across all
    languages with more information.
    • C++ - NxException Object
    • Python - NxException Object
Note:
By default, simple exception is enabled for an App. If the App supports Advanced Exception handling then to enable Advanced exception, set advException flag to TRUE in getSdkInst Api.
