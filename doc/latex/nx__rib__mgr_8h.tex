\subsection{Detailed Description}
Abstract SDK interface to Nexus(NXOS) Layer 3 Unicast RIB. Provides necessary Abstraction/Plugin for NXOS RIB functionalities. It provides the Ability to register for Route and VRF updates and more.

Steps to register for route event from RIB: \par
 1) Instantiate NX-\/SDK Rib Manager Object using getRibMgr(). \par
 2) To watch for VRF updates, use watchVrf() (or) watchAllVrf() for all VRF updates. \par
 3) To watch for L3 route updates, use watchL3Route() \par
 4) Register for callback handler with NxRibMgr to get the regsitered RIB updates. \par
 5) Overload NxRibMgrHandler class and its member functions like \par
 (postVrfCb() -\/ To get watched VRF updates, \par
 postL3RouteCb() -\/ To get watched L3 route update, \par
 etc) with your own implementation to handle various events.\par
 6) NOTE: Adding/Deleting Routes in Rib will be supported soon. \par
