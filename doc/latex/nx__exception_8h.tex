\subsection{Detailed Description}
Abstract Exception Interface for all thrown NX errors. NX-\/SDK v1.0.0 supports \par

\begin{DoxyItemize}
\item Throws Simple exception of the type in \par

\begin{DoxyItemize}
\item C++ -\/ std::logic\_\-err \par

\item Python -\/ String Object. \par
\par

\end{DoxyItemize}
\end{DoxyItemize}

NX-\/SDK v1.5.0 supports \par

\begin{DoxyItemize}
\item Throws Simple exception of the type as version v1.0.0, \par

\begin{DoxyItemize}
\item C++ -\/ std::logic\_\-err \par

\item Python -\/ String Object. \par
\par

\end{DoxyItemize}
\end{DoxyItemize}


\begin{DoxyItemize}
\item Also Throws Advanced exeception of the type NxException across all \par
 languages with more information. \par

\begin{DoxyItemize}
\item C++ -\/ NxException Object \par

\item Python -\/ NxException Object \par

\end{DoxyItemize}
\end{DoxyItemize}

\begin{DoxyNote}{Note}
By default, simple exception is enabled for an App. If the App supports Advanced Exception handling then to enable Advanced exception, set advException flag to TRUE in getSdkInst Api.
\end{DoxyNote}
