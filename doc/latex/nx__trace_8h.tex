\subsection{Detailed Description}
Abstract Tracer interface to generate custom Syslogs, Events \& Errors history in Nexus Switches. Steps to construct a custom syslog, Events \& Erros history: 1) Instantiate NxSDK Tracer Object using getTracer(). 2) To generate custom syslogs, use TracerObj-\/$>$syslog(..). 3) To generate custom events, use TracerObj-\/$>$event(...). To check the events, use \char`\"{}show $<$AppName$>$ internal event-\/history events\char`\"{} 4) To generate custom errors, use TracerObj-\/$>$error(...). To check the events, use \char`\"{}show $<$AppName$>$ internal event-\/history events\char`\"{}( 5) Refer to examples for usage.
